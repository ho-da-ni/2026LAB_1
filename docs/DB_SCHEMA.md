# DB_SCHEMA.md Template

## 문서 목적
DB 메타데이터로 확인 가능한 테이블, 컬럼, 제약 조건을 기록한다.

## 작성 원칙

- 스키마 정보는 DB 메타데이터에서 확인된 값만 적는다.
- 컬럼 타입, nullable, key 정보는 추정하지 않는다.
- 애매한 관계는 `UNKNOWN` 또는 `needs_review`로 남긴다.
